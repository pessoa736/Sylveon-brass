# Sylveon-brass

Um modpack focado na fusão entre automação industrial pesada, exploração de dimensões e a jornada Pokémon definitiva. 

> **Aviso de Linha Editorial:** Este pack não tem nenhuma intenção de ser o mais balanceado ou o "melhor" modpack técnico do mundo. Ele foi criado puramente para um grupo de amigos jogarem em um servidor sem muito planejamento, apenas pela diversão e pelo caos de misturar fábricas com Pokémon.

---

[imagem ilustrativa]

[imagem ilustrativa]

</details>

<details>
  <summary> ℹ️ <b>Instalação Manual, Compilando à mão.</b> </summary>
  
  Para quem quer buildar o modpack direto do código-fonte usando o **Packwiz CLI**:
  
  **1.** Clone este repositório na sua máquina:
     ```bash
     git clone [https://github.com/pessoa736/Sylveon-brass.git](https://github.com/pessoa736/Sylveon-brass.git)
     ```
     
  **2.** Certifique-se de ter o executável do ```packwiz``` instalado.
  
  **3.** Para exportar o modpack no formato do CurseForge ou Modrinth, rode no terminal:
  ```Bash
    packwiz curseforge export
    # ou
    packwiz modrinth export
  ```
  
  **4.** Pegue o arquivo ```.zip``` ou ```.mrpack``` gerado e arraste para o seu launcher de preferência.
</details>
